# ⏰ SmartPonto

Sistema mobile de controle de ponto desenvolvido em Flutter.

O SmartPonto foi criado com o objetivo de facilitar o registro e gerenciamento de horários de entrada e saída de funcionários, oferecendo uma interface simples, moderna e prática.

# ▶️ Como Executar

## Clone o repositório

```bash
git clone https://github.com/GabryePatrickSoares/SmartPonto.git
```

## Acesse a pasta

```bash
cd SmartPonto
```

## Instale as dependências

```bash
flutter pub get
```

## Execute o projeto

```bash
flutter run
```

---

# 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com fins de aprendizado e prática no desenvolvimento mobile utilizando Flutter, focando na criação de interfaces modernas, lógica de controle de horários e experiência do usuário.
